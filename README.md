# Trello clone with NextJS
## Structure 
Project structure should be:
* **app** $\rightarrow$ File-system based routes
* **components** $\rightarrow$ Components and widgets
* **database** $\rightarrow$ DB connection & querying
* **emails** $\rightarrow$ HTML email templates
* **functions** $\rightarrow$ React server functions
* **lib** $\rightarrow$ Cross-cutting utilities
* **public** $\rightarrow$ Images and static assets
